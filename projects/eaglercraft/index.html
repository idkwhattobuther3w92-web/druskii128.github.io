How to Play Eaglercraft (Minecraft Browser) Unblocked

Eaglercraft is an open-source browser port of Minecraft Java Edition that lets you play the iconic sandbox game with no download. Build structures, explore worlds, and survive monsters entirely in your browser window. Our fast GitHub mirror keeps it unblocked and accessible from school networks and Chromebooks.

Controls

  • WASD — move
  • Space — jump / swim up
  • Left Click — break blocks / attack
  • Right Click — place blocks / use item
  • E — open inventory
  • Esc — pause
